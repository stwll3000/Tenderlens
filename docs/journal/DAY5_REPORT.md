- **Решение:** Отложено, работаем с JSON-файлами
- **Статус:** Требуется обновление connection string в Supabase Dashboard

- **Причина:** Сложная структура HTML таблиц
- **Решение:** Требуется доработка парсера
- **Статус:** Отложено на День 6

### 3. Суббота — самый популярный день?
