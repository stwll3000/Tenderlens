## 🚀 Как запустить

### 1. Установить зависимости
```bash
pip install python-telegram-bot==20.7
```

### 2. Создать бота через @BotFather
1. Откройте https://t.me/BotFather
2. Отправьте `/newbot`
3. Укажите имя и username
4. Скопируйте токен

### 3. Обновить .env
```env
TELEGRAM_BOT_TOKEN=ваш-токен-здесь
```

### 4. Запустить
```bash
python bot/main.py
```

### 5. Протестировать
Откройте бота в Telegram и попробуйте команды!
