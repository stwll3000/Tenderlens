# TenderLens Dashboard

Интерактивный веб-дашборд для анализа рынка государственных закупок России.

## Возможности

### 📊 Основные метрики
- Общее количество лотов
- Суммарный объём закупок
- Средняя цена закупки
- Количество уникальных заказчиков

### 🔍 Фильтры
- По регионам (Новосибирская область, Москва, Московская область)
- По законам (44-ФЗ, 223-ФЗ)
- По диапазону цен (слайдер)
- По статусам закупок

### 💰 Анализ цен
- Статистика распределения (мин, макс, медиана, среднее)
- Гистограмма распределения цен
- Box plots по регионам
- Коэффициент вариации

### 🏆 Анализ конкуренции
- Топ-10 заказчиков по количеству закупок
- Топ-10 заказчиков по объёму закупок
- Индекс концентрации рынка (HHI)
- Доля топ-5 и топ-10 заказчиков

### 📈 Визуализация
- Интерактивные графики (Plotly)
- Распределение по законам и статусам
- Сравнение регионов
- Pie charts и bar charts

### 📋 Экспорт данных
- Просмотр таблицы с фильтрацией
- Экспорт в CSV

## Установка зависимостей

```bash
pip install streamlit plotly pandas
```

## Запуск

```bash
streamlit run dashboard/app.py
```

Дашборд откроется в браузере по адресу: `http://localhost:8501`

## Структура

```
dashboard/
├── app.py          # Главный файл приложения
└── README.md       # Документация
```

## Используемые модули

- `analytics/pricing.py` — анализ цен
- `analytics/competition.py` — анализ конкуренции
- `data/lots_all_*.json` — данные о закупках

## Технические детали

- **Framework:** Streamlit
- **Визуализация:** Plotly Express, Plotly Graph Objects
- **Данные:** JSON (автоматический поиск последнего файла)
- **Кэширование:** `@st.cache_data` для быстрой загрузки

## Вкладки дашборда

1. **💰 Анализ цен** — статистика и распределение цен
2. **🏆 Конкуренция** — топ заказчиков и концентрация рынка
3. **📊 Распределения** — графики по законам, статусам, регионам
4. **📋 Данные** — таблица с возможностью экспорта

## Примеры использования

### Анализ конкретного региона
1. Выберите регион в боковой панели
2. Посмотрите изменение метрик
3. Изучите топ заказчиков региона

### Сравнение 44-ФЗ и 223-ФЗ
1. Переключайтесь между законами в фильтре
2. Сравните средние цены и объёмы
3. Проанализируйте распределение статусов

### Поиск крупных закупок
1. Установите минимальную цену через слайдер
2. Отсортируйте таблицу по цене
3. Экспортируйте результаты в CSV

## Производительность

- Загрузка 600 лотов: ~1 секунда
- Применение фильтров: мгновенно
- Построение графиков: ~0.5 секунды
