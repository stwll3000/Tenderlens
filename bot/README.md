# TenderLens Telegram Bot

Telegram-бот для анализа и мониторинга госзакупок России.

## Возможности

- 📊 **Статистика** — общая статистика по закупкам
- 🔍 **Поиск** — поиск закупок по ключевым словам
- 🎯 **Топ ниш** — анализ перспективных регионов
- 🔔 **Алерты** — уведомления о новых закупках (в разработке)

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте бота через [@BotFather](https://t.me/BotFather) в Telegram:
   - Отправьте `/newbot`
   - Укажите имя бота (например, `TenderLens Bot`)
   - Укажите username (например, `tenderlens_bot`)
   - Скопируйте токен бота

3. Добавьте токен в `.env` файл:
```env
TELEGRAM_BOT_TOKEN=your-bot-token-here
TELEGRAM_ADMIN_ID=your-telegram-user-id
```

4. Убедитесь, что есть данные в директории `data/`:
```bash
ls data/lots_*.json
```

## Запуск

```bash
python bot/main.py
```

Или через модуль:
```bash
python -m bot.main
```

## Команды бота

- `/start` — Приветствие и инструкции
- `/help` — Справка по командам
- `/stats` — Статистика по закупкам
- `/search <запрос>` — Поиск закупок
- `/top_niches` — Топ-5 перспективных ниш

## Примеры использования

### Поиск закупок
```
/search строительство
/search медицинское оборудование
/search ремонт дорог
```

### Статистика
```
/stats
```

Показывает:
- Общее количество лотов
- Общий объем закупок
- Среднюю и медианную цену
- Количество регионов и заказчиков
- Распределение по законам (44-ФЗ, 223-ФЗ)

### Топ ниш
```
/top_niches
```

Показывает топ-5 регионов по объему закупок.

## Конфигурация

Настройки в `bot/config.py`:

- `MAX_RESULTS_PER_PAGE` — максимум результатов на страницу (по умолчанию 10)
- `ALERT_CHECK_INTERVAL` — интервал проверки новых закупок (3600 сек = 1 час)
- `MIN_PRICE_FOR_ALERT` — минимальная цена для алерта (1 млн ₽)

## Логирование

Логи сохраняются в `logs/bot.log` и выводятся в консоль.

## Troubleshooting

### Ошибка: "TELEGRAM_BOT_TOKEN не установлен"
Добавьте токен в `.env` файл:
```env
TELEGRAM_BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
```

### Ошибка: "Нет данных для работы бота"
Убедитесь, что в директории `data/` есть JSON-файлы с лотами:
```bash
python scraper/collect_multi_regions.py
```

### Бот не отвечает
1. Проверьте, что бот запущен
2. Проверьте токен в `.env`
3. Проверьте логи в `logs/bot.log`
