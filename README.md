## Быстрый старт

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Настройка базы данных

Следуйте инструкции в `SETUP_DATABASE.md`:

1. Создайте проект в [Supabase](https://supabase.com)
2. Скопируйте строку подключения (Database URL)
3. Создайте `.env` файл:
   ```bash
   copy .env.example .env
   ```
4. Добавьте DATABASE_URL в `.env`

### 3. Инициализация БД и загрузка данных
```bash
python db/init_database.py
```

Этот скрипт автоматически:
- Проверит подключение к PostgreSQL
- Создаст таблицы (regions, customers, lots)
- Загрузит справочник регионов
- Загрузит 150 собранных лотов

### 4. Сбор новых данных
```bash
python scraper/fetch_lots.py
```

### 5. Валидация данных
```bash
python scraper/validate_data.py
```
