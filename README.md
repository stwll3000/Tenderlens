# TenderLens — Аналитическая платформа рынка госзакупок РФ

Платформа для анализа данных о государственных закупках России с целью выявления перспективных ниш и аномалий в торгах.

## Установка и запуск

### 1. Клонировать репозиторий
```bash
git clone <repo-url>
cd Tenderlens
```

### 2. Установить зависимости
```bash
pip install -r requirements.txt
```

### 3. Настроить конфигурацию
```bash
cp .env.example .env
# Отредактировать .env при необходимости
```

### 4. Запустить парсер
```bash
python scraper/fetch_lots.py
```

### 5. Валидировать данные
```bash
python scraper/validate_data.py
```
