# 🚀 Быстрый старт TenderLens Dashboard

## Запуск дашборда

```bash
streamlit run dashboard/app.py
```

Дашборд откроется в браузере: `http://localhost:8501`

## Что внутри

- **600 лотов** из 3 регионов
- **8 интерактивных графиков** (Plotly)
- **4 фильтра** (регион, закон, цена, статус)
- **4 вкладки** анализа (цены, конкуренция, распределения, данные)
- **Экспорт в CSV**

## Возможности

### Анализ цен
- Статистика распределения
- Гистограмма и box plots
- Выявление выбросов

### Анализ конкуренции
- Топ-10 заказчиков
- Индекс концентрации рынка (HHI)
- Доля крупнейших игроков

### Визуализация
- Распределение по законам (44-ФЗ vs 223-ФЗ)
- Статусы закупок
- Сравнение регионов

## Тестирование

```bash
python dashboard/test_dashboard.py
```

Проверяет:
- Наличие данных
- Работу модулей аналитики
- Установку зависимостей
